## Installation

MERN-Express-Stripe requires [Node.js](https://nodejs.org/) or [Yarn](https://yarnpkg.com/) to run.

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## Features

Features included inside the project

- Stripe Checkout
- Stripe Payment
- ENV's
- Cors
- UUID
